# Next-Up

Next.js boilerplate preconfigured with redux, next-routes, sass, postcss, polyfills, various components and tools, gzip compression, and rg-cli.

## System Requirements

- Node.js v10.6+

## Installation

- download repo
- run `npm install`

## Development

- `npm run dev`
- go to `http://localhost:3000` from your browser

## Production

- run `npm run build`
- run `npm start`
- go to `http://localhost:3000` from your browser
